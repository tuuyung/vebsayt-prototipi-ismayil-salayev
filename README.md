Layihənin əsas mövzusu universistet tələbələri üçün nəzərdə tutulmuş forum(söhbət) saytının login və qeydiyyat stukturudur. Buraya login formu, database, form ilə database-i əlaqələndirən backend (runtime environment) və veb səhifənin internetdə görünməsini və database-in işləkliyini təmin edən server və domen elementləri daxildir. Bu zaman istifadə olunan nginx kimi alətlər və Cloudflare kimi domen və təhlükəsizlik həllərindən istifadə olunacaq. Veb səhifənin təhlükəsizliyi təməl səviyyədə, habelə şifrələrin nginx root yolundan kənarda dotenv daxilində saxlanılması kimi üsullarla təmin olunacaq. İstifadəçi şifrələrinin hash alqoritmləri ilə saxlanması və autentifikasiya zamanı hash qarşılaşdırması kimi kriptoqrafik təhlükəsizlik həlləri layihənin pilot versiyasında əhatə olunmayacaq. Server layihədən kənar potensial təhdidlərin kənarlaşdırılması üçün LXC virtual maşın mühitində işləyəcək.
